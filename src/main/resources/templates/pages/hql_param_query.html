使用HQL进行查询(带参数)
               select st  from    Model st     where   st.id=:id   order    by st.id
使用HQL进行查询(带多参数)
               select st  from    Model st     where   st.id=:id and st.name=:name  order    by st.id
使用HQL进行查询(关联查询)
               select st.homeWorkList from Student st where st.userInfo.name=:name
